# Support and SLA Template

This is an operational template, not a guarantee.

## Included support for the local reference product
- installation issues
- deterministic replay issues
- benchmark execution issues

## Not included
- hosted production uptime
- security incident response
- custom model integration support without a separate agreement

## Example response targets
- blocking install issue: next business day
- benchmark question: two business days
- feature request triage: weekly batch
