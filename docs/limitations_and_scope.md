# Limitations and Scope

## What this bundle is

- deterministic local reference product
- benchmarked research artifact
- archive-ready source snapshot
- sales/demo package

## What this bundle is not

- hosted multi-tenant service
- audited security product
- latency-optimized production system
- customer-validated commercial deployment

## Technical limits

- single-process local JSON persistence
- no network API server
- no authn/authz layer
- no distributed consistency model
- benchmark claims are spec-level and local-reference-level only

## Claim discipline

Do not state or imply:
- hosted production readiness
- external security certification
- customer adoption proof
- benchmark victory by an unvalidated external kernel
