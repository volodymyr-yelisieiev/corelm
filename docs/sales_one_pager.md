# Core LM Sales One-Pager

## What it is

Core LM is a deterministic memory architecture that separates text generation
from truth storage. LLM output is treated as excitation, while durable state,
provenance, and replay live in the core system.

## Problem

Raw context windows forget, mix branches, and cannot explain why a fact is
current. Enterprises need memory that is inspectable and replayable.

## What buyers get

- deterministic structured memory
- append-only provenance ledger
- contradiction supersession instead of silent overwrite
- branch isolation
- frozen benchmark with baseline comparisons

## Proof in this bundle

- formal state-space specification
- deterministic reference kernel
- oracle benchmark ceiling
- adversarial and real-text scenarios
- local product demo via CLI

## Best fit buyers

- research teams building long-horizon agents
- infra teams evaluating memory backends
- founders validating memory claims before production buildout

## Commercial posture

This bundle is suitable for demos, evaluation licenses, and design-partner
conversations. Hosted production commitments require separate validation.
